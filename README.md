# ✈️ Travel UI Experience

Modern, responsive travel discovery web interface inspired by platforms like Airbnb, designed to deliver a seamless and visually engaging user experience.

---

## 🚀 Features

* 🔍 Smart search bar with location input
* 🧭 Category-based browsing (beaches, mountains, cities, etc.)
* 🏨 Interactive travel cards with images and details
* 🎨 Smooth animations and transitions
* 📱 Fully responsive design (mobile + desktop)
* 🌙 Clean and modern UI with intuitive navigation

---

## 🛠 Tech Stack

* React
* TypeScript
* Tailwind CSS
* Framer Motion (animations)
* Vite

---

## 🎯 Project Goal

To design and develop a high-quality frontend interface that mimics real-world travel platforms, focusing on usability, performance, and modern UI/UX principles.
